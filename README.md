# How to run a local `Flask Api App`
1. Create virtualenv
```bash
$ virtualenv -p python3 .venv
```

2. Enter into your virtualenv
```bash
$ source .venv/bin/activate
(.venv) $
```

3. Install requirements
```bash
(.venv) $ pip install -r requirements.txt
```

4. Run Flask App
```
(.venv) $ python app.py
```
